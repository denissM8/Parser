### Simple guide
1) Create Folder TRANFER_PARSER
2) In TRANFER_PARSER folder create TRANSFER_ERRORS.txt file
3) Create or add .bat and .ps1 files
4) Add errors from chat to TRANSFER_ERRORS.txt file
6) Start .bat file
7) Open FORMATED_TRANSFERS.csv file and COPY fields
8) Open Excel file and choose 4xN fields -> than PASTE formated transfers
